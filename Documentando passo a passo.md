# Documentação

## Para iniciar o projeto:
npm init -y

## Instalar o express
npm install express --save
--save serve para adicionar as dependencias do pacote express no seu package.json

## Instalar o nodemon para permitir hot reload
npm install nodemon -D
-D é para habilitar o modo de desenvolvedor

## Executando o servidor
node ${arquivo}.js
npm run dev (parametro dev foi definido no package.json)

## Instalar o BD
npm install mysql

(Sugestão: "DBeaver Community" ao invés do MySQL Workbenck)
